# Research Report Template

## Suggested Flow

1. **Understand** — Define the research question and scope
   - What exactly are we trying to learn?
   - What's the scope (breadth vs depth)?
   - What format should the output take?
   - Are there deadlines or resource constraints?

2. **Research** — Find, collect, and organize information
   - Identify key sources (web, papers, docs, people)
   - Collect relevant data, quotes, findings
   - Track sources for citation
   - Note contradictory information
   - Flag gaps and unknowns

3. **Plan** — Structure the report
   - Organize findings by theme, chronology, or importance
   - Decide on report structure (exec summary, body, appendices)
   - Identify the key narrative or argument

4. **Draft** — Write the report
   - Start with the executive summary / key findings
   - Build out each section with evidence
   - Include data visualizations if appropriate
   - Acknowledge gaps and limitations

5. **Review** — Check against success criteria
   - Does it answer the original research question?
   - Are sources credible and cited?
   - Is the structure clear and logical?
   - Are limitations acknowledged?
   - Is it actionable?

6. **Finalize** — Polish and deliver
   - Proofread and format
   - Ensure citations are consistent
   - Deliver in requested format

## Research-Specific Success Criteria

- [ ] Research question is specific and answerable
- [ ] At least 3-5 credible sources consulted
- [ ] Findings directly address the question
- [ ] Contradictions and uncertainties are noted
- [ ] Recommendations are clear and actionable

## Tips

- Research often takes longer than expected — set time limits per source
- Not everything needs to be in the final report; be ruthless about cutting
- Prioritize primary sources over secondary summaries
- Start organizing findings early (during research, not after)
- If you can't find information, say so — it's a valid finding
